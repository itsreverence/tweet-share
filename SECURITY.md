# Security Policy

## Reporting vulnerabilities

Please report vulnerabilities in this userscript through GitHub private vulnerability reporting or by opening an issue if the details are not sensitive.

Do not paste live Discord webhook URLs, private/protected post contents, account cookies, or X/Twitter session details into public issues. If a report requires sensitive details, use a private channel.

## Scope

In scope for this repository:

- Userscript code vulnerabilities
- Build or release workflow vulnerabilities
- Unsafe webhook handling or storage behavior in the userscript
- Documentation that could cause unsafe installation or credential handling

Out of scope:

- X/Twitter platform vulnerabilities
- Discord platform vulnerabilities
- Lost, leaked, or abused webhook URLs outside this userscript
- Requests to bypass X/Twitter access controls or scrape content the user cannot view
